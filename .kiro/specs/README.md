# Specs Index

## Active Spec

- `mvp-lead-ops-demo` — Demo Core proving pasted text/transcript input to draft lead, human review, dashboard lead management, follow-ups, basic metrics, and audit/activity.

## Deferred Specs

- `copilotkit-assistant` — dashboard assistant search, calendar awareness, and mutation previews.
- `custom-fields` — workspace-specific field definitions and lead values.
- `calendar-view` — internal lead-related calendar display and availability checks.
- `telegram-ingestion` — Telegram text and voice-note intake.
- `website-lead-capture-widget` — Nuxt/laptop-repair site capture flow into this system.
- `telephony-recording-ingestion` — real call recording/transcript source.
- `analytics-and-reports` — charts, grouping, reporting, and product instrumentation.

Only the active spec should drive the first implementation session. Later MVP tasks inside the active spec are planning context, not implementation tasks until promoted or explicitly requested.
